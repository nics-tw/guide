#### CSS

- `.button`：作為全部按鈕的基底。
- `.button-primary`：作為主要或正面的動作視覺傳達。
- `.button-danger`：作為有破壞性（如刪除資料）或負面的動作視覺傳達。

#### 親和力

按鈕 CSS classes 不能使用在 `<a>` 上，因為 `<button>` 和 `<a>` 各有不同的使用、啟動方式，因此必須在視覺上做明顯的區分以便使用者分辨。

%}

#### CSS

- `[disabled]`, `.button-disabled`：作為不可用的按鈕視覺。

#### 親和力

- **可用於**剛送出的表單按鈕，以用於避免二次送出。
- **不可用於**驗證錯誤的表單按鈕。
- **不可用於**尚未填寫完的表單按鈕。
